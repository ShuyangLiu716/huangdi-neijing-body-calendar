# 黄帝内经身体日历

**你查天气决定穿什么，查日历决定做什么。这个日历，决定你今天怎么对待身体。**

一份结合《黄帝内经》、现代运动科学、脑科学和你所在城市天气的每日身体指南。每天打开，照着做就行。

## 修炼体系

这个日历不只是"看一眼就关掉"。它有一套轻量的修炼系统，让"关注身体"变成一种习惯。

### 节气修炼

二十四节气 = 24 个修炼节点。

每个节气有专属的修炼重点：

- **芒种**：清热祛湿，微动微汗
- **冬至**：藏精蓄锐，早睡晚起
- **春分**：舒展筋骨，踏青散步
- **白露**：润燥养肺，收敛心神

跟着节气走，一年下来，你会对身体的节奏有全新的感知。

### 身体感知称号

每天打开日历 + 回答内观五问 = 积累「感知值」。

感知值不跟任何人比，只跟你自己比。达到一定天数，解锁称号：

| 称号 | 天数 | 含义 |
|------|------|------|
| **初觉** | 1-7 天 | 刚开始关注身体 |
| **渐醒** | 8-21 天 | 身体开始有回应 |
| **入微** | 22-60 天 | 能感知细微变化 |
| **自在** | 60 天+ | 身体成为日常，不刻意也能关注 |

**没有排行榜，没有惩罚，断签不扣分。**

你只是每天花 2 分钟，问自己一句："今天身体怎么样？"

## 这个日历的边界

**会做的事：**
- 给出今天的作息、饮食、运动、情志建议
- 结合节气、天气、你的身体情况调整建议
- 提供非诊断性的身体自查问题
- 标注每条建议的来源依据

**不会做的事：**
- 不诊断疾病（"你这是XXX病"）
- 不开药方（"吃这个中药"）
- 不推荐针灸、艾灸、拔罐、刮痧操作
- 不做吉凶判断（"今天宜嫁娶"）
- 不承诺治疗效果（"坚持做就能治好"）

如果出现胸痛、严重呼吸困难、高烧、急性疼痛等症状，请直接就医，不要用这个替代诊疗。
