# Daily Calendar Template

Use this template for full daily outputs. Keep it concise unless the user asks for a detailed plan.

```markdown
# 中医身体日历｜YYYY-MM-DD

位置：成都，四川（如用户另给地点则替换）

## 今日主题
[用一句话连接成都本地气候、季节/节气、昼夜节律和《内经》思想。避免吉凶判断。]

## 今日宜
- [1-3 条正向行为，例如早见自然光、饭后散步、晚间收心。]

## 今日不宜
- [1-3 条行为提醒，例如熬夜、久坐不动、空腹高强度训练。不要写玄学禁忌。]

## 作息
[具体到早/午/晚的作息建议。包含睡眠、光照、久坐打断。成都多云雾、少日照时，建议抓住上午户外光线窗口；雨天则给室内替代。]

## 饮食
[食物类别、进食节制、饮水、晚餐时间。成都湿热或阴雨时，强调清淡、规律、少冰冷油腻。不给药膳处方，不承诺治疗。]

## 运动
[给出强度、时间、形式、替代方案。默认中低强度，提醒按体感调整。成都雨天、湿热或空气闷时，提供室内走动、关节活动、轻力量等替代方案。]

## 情志
[1-2 条情绪调节或呼吸放松建议，避免心理治疗化承诺。]

## 内观五问
1. [睡眠/精神]
2. [食欲/消化]
3. [疼痛/不适]
4. [运动后反应]
5. [压力/情绪]

## 风险提醒
[列出今天相关的红旗提醒；没有明显风险也要说明不适时暂停并就医。]

## 依据
- 《黄帝内经》相关思想：[素问/灵枢/四时养生/起居有常等，按实际使用写。]
- 中医养生规范：[国家中医药管理局养生保健服务规范，限定为养生教育。]
- 现代运动科学：[WHO/ACSM 相关指南，说明用于运动和久坐建议。]
- 成都本地气候：[中国气象局/四川省政府/中国天气网/中央气象台，说明是否为实时天气或区域气候参考。]
```

## Tone

- Use calm, practical, non-mystical language.
- Keep "今日宜/不宜" behavioral and health-oriented.
- Prefer "建议", "可", "避免", "如有不适" over absolute commands.
- Do not over-explain the entire theory unless the user asks.

## Personalization Slots

If the user gives personal context, adapt these sections:

- Sleep issue: adjust 作息, 情志, caffeine timing, evening exercise intensity.
- Pain or injury: adjust 运动 and 风险提醒 first.
- Chronic disease or pregnancy: keep advice general and recommend professional assessment before changing exercise.
- Training goal: separate wellness calendar from performance training advice.
