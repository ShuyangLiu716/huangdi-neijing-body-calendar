# 用户画像

> 填写后，每日日历会自动根据这些信息生成。
> 用户在对话中提供的信息会覆盖这里的默认值。

---

## 基本信息

city: 成都
age: ~
language: 中文

## 身体情况

health_notes: ~
# 填写你的身体情况，例如：
# - 肩颈僵硬，久坐办公
# - 产后3个月
# - 膝盖有旧伤
# - 睡眠质量差
# 留空则生成通用日历

## 输出格式

output_format: full
# full     = 完整10模块日历（默认）
# concise  = 精简版：主题 + 宜忌 + 运动 + 饮食（4个模块）
# weibo    = 小红书/微博短文案版（300字以内）

## 输出目录（可选）

output_dir: ~
# 填写本地路径，日历会自动写入该目录
# 例如：
# - ~/Documents/body-calendar/
# - D:/Obsidian/每日身体日历/
# - ./output/
# 留空则只在终端显示
