# Source Policy

Use first-hand or near-first-hand sources whenever possible. If a source cannot be checked in the current turn, label it as a reference category rather than pretending to quote it exactly.

## Source Priority

1. **Classical source layer**
   - Huangdi Neijing primary text: `素问` and `灵枢`.
   - Prefer scanned editions, reputable punctuated editions, museum/library records, or stable classical text databases.
   - Useful reference entry points:
     - 中国国家博物馆藏《灵枢·素问》说明: https://www.chnmuseum.cn/zp/zpml/gjwxbt/202203/t20220301_253962_wap.shtml
     - 中國哲學書電子化計劃《灵枢经》: https://ctext.org/huangdi-neijing/ling-shu-jing/zhs
   - Use these sources for themes such as seasonal adaptation, moderation, sleep-wake rhythm, emotional regulation, and harmony with natural cycles.

2. **Modern TCM public-service layer**
   - Official policy or service-scope documents from state or professional bodies.
   - Core boundary source:
     - 国家中医药管理局《中医养生保健服务规范（试行）》, published through 中国政府网: https://www.gov.cn/zhengce/zhengceku/2023-05/08/content_5754578.htm
   - Use this layer to define wellness education scope and avoid clinical diagnosis, treatment, prescriptions, and invasive operations.

3. **Modern exercise science layer**
   - WHO physical activity and sedentary behavior guidance:
     - WHO 2020 Guidelines on physical activity and sedentary behaviour: https://www.who.int/publications/b/55518
     - WHO Physical activity fact sheet: https://www.who.int/news-room/fact-sheets/detail/physical-activity%E2%80%AF%E2%80%AF
   - ACSM-style exercise screening and prescription references:
     - ACSM Guidelines for Exercise Testing and Prescription, current edition if available to the user or institution.
     - ACSM preparticipation health screening materials when risk, chronic disease, symptoms, or exercise intensity decisions matter.
   - Use this layer for weekly activity targets, sedentary breaks, intensity descriptions, gradual progression, and symptom-based caution.

4. **Local meteorology layer for Chengdu**
   - Default location: Chengdu, Sichuan, unless the user says otherwise.
   - Official climate/weather entry points:
     - 中国气象局关于成都气候特点的说明: https://www.cma.gov.cn/ztbd/2023zt/20230712/2023071304/202307/t20230727_5673807.html
     - 四川省人民政府气候资源说明: https://www.sc.gov.cn/10462/10464/10757/10868/2018/4/17/10449170.shtml
     - 中国天气网成都天气预报: https://www.weather.com.cn/weather1dn/101270101.shtml
     - 中央气象台成都天气: https://m.nmc.cn/publish/forecast/ASC/chengdu.html
   - Use this layer for local context such as Sichuan Basin humidity, cloud/fog, low sunshine, rain-season damp heat, and outdoor activity feasibility.
   - Do not treat climate normals as today's live weather. For today's rain, temperature, air quality, or humidity, check current weather if tools are available.

## Citation Rules

- Include an "依据" section in every full daily calendar.
- Prefer source categories plus links over long quotations.
- Quote classical text only when the exact wording has been verified in the current context. Otherwise paraphrase and name the chapter/source category.
- Do not cite unsourced folk claims, social media posts, or commercial wellness pages as authority.
- When classical and modern exercise guidance differ, choose the more conservative modern safety interpretation.
- When Chengdu climate guidance is used without live weather lookup, say it is based on regional climate characteristics, not a real-time forecast.

## Evidence Wording

Use wording like:

- "《黄帝内经》四时养生思想：用于解释今日作息与情志方向。"
- "国家中医药管理局养生保健服务规范：用于限定本建议为养生教育，不替代诊疗。"
- "WHO 身体活动指南：用于运动时长、久坐打断和循序渐进建议。"
- "成都本地气候参考：中国气象局/四川省政府资料显示成都及四川盆地具有湿度大、多云雾、少日照等特点，因此今日建议保留室内活动替代方案。"

Avoid wording like:

- "黄帝内经证明此法可以治疗..."
- "按今天气运必然..."
- "此食疗方可治..."
